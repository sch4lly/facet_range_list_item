INTRODUCTION
------------

This Facet Range list Item module provides the facets with dynamic range. The
possible values this field can contain. Enter one value per line, in the format
key|label.

For example.
0-20|0-20 Minutes.
20-40|20-40 Minutes
40-999|40+ Minutes

For a full description of the module, visit the project page:
https://www.drupal.org/project/facet_range_list_item


REQUIREMENTS
------------

This module requires the following modules:

Facets (https://www.drupal.org/project/facets)
Search API (https://www.drupal.org/project/search_api)


INSTALLATION
------------

Install the Facet Range list Item module as you would normally install a
contributed Drupal module. Visit https://www.drupal.org/node/1897420 for further
information.


CONFIGURATION
-------------

    1. Navigate to Administration > Configuration > Search and Metadata > Facets
       Add facet (field type => number)
    2. On Edit Facet page -> Range List Item Processor -> Enable checkbox ->
          Enter Range -> Save Configurations.
