Facet Range list Item
-----------------------
This module provides an option for user-specific range as the processor for Search API

REQUIREMENTS
------------
No other modules required; we're supporting Drupal Core's search as a source for
creating facets.

INTRODUCTION
------------
The possible values this field can contain. Enter one value per line, in the format key|label.
The key must be numeric range. The label will be used in display values of facet.

For example.
0-20|0-20 Minutes.
20-40|20-40 Minutes
40-999|40+ Minutes
